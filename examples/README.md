# enDjinn examples

Currently two examples are proved. The bare bones [hello_enDjinn](./hello_enDjinn/) 
that is a 'get a rotating sprite on screen in as fewer than 100 lines, and the more involved 
[mode_orchestration](./mode_orchestration/) that adds interactivity while demonstrating the controller 
system and also tries to demonstrate the built in state handling in the enj_mod_% subsystem.
